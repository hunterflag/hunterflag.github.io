innerHTML、innerText、textContent、outerHTML、outerText 屬性的差異

keywords: innerHTML、innerText、outerHTML|、outerText、textContent

摘要

inner outer
HTML innerHTML outerHTML
Text innerText outerText
textContent
  • innerHTML: 該元素的全部後代, 的原始內容文字串, 包括 <tag>、與 文字本身
  • textContent: 是 innerHTML 的內容文字串, 拿掉 <tag> 之後的剩餘文字
  • innerText: 是 innerHTML 的內容文字串, 拿掉不可視元素, 再拿掉可視元素的 <tag> 之後的剩餘文字
  • outerHTML: 是保留該元素本身的內容, 再加上 innerHTML
  • outerText: 是 outerHTML 的內容文字串, 拿掉不可視元素, 再拿掉可視元素的 <tag> 之後的剩餘文字
